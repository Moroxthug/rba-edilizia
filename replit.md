# Project Overview

This is a website project. All development builds on top of the established design foundation.

## User Preferences

- **Design is locked**: Font, layout, colors, buttons, and all visual styles of the homepage must remain intact at all times. Never change or override existing design decisions.
- **Build on top**: New features and pages must adopt the existing design language — colors, typography, spacing, button styles — without introducing new conflicting styles.
- **Language**: The user communicates in Italian. Respond in Italian unless they switch languages.
